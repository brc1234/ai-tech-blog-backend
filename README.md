**AI Tech Blog - Backend**

Ich habe dieses Projekt als eine sichere API-Infrastruktur entwickelt, die die Datenverwaltung, die Blogbeiträge und die Kontaktnachrichten für den AI Tech Blog steuert.

**Projektdetails:**

* Ich habe **Node.js** und **Express.js** für den Aufbau des Servers genutzt.
* Ich habe **MongoDB** und **Mongoose** für die Datenbankverbindung verwendet.
* Ich habe die Datenbankverbindung mit einer **.env** Datei verschlüsselt und abgesichert.
* Es gibt API-Routen für das **Abrufen** und **Hinzufügen** von Blogbeiträgen.
* Es gibt eine API-Route zum **Speichern** von Nachrichten aus dem Kontaktformular.
* Ich habe **CORS** integriert, um eine sichere Kommunikation mit dem Frontend zu ermöglichen.

**Installation und Start:**

* Zuerst die Pakete mit dem Befehl **npm install** installieren.
* Eine **.env** Datei im Hauptordner erstellen und die MONGO_URI Variable definieren.
* Danach den Server mit dem Befehl **npm start** oder **node server.js** starten.
